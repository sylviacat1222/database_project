#### 12. index.html: 首頁、搜尋功能1(由地點搜尋)頁面1(以縣市、鄉鎮市區搜尋)

#### 17. search_comment.html: 搜尋留言頁面1(以景點關鍵字搜尋留言)
#### 18. update_form.php: 更新留言頁面1(更新留言的表格)
#### 19. weather_form.html: 搜尋功能2(由天氣條件、日期搜尋)頁面1(表格)
